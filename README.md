## 04. Overview do Retrofit:
* Trabalhar com maior simplificade com nosso código.
* Utilizaremos a API padrão do GitHub.
* Retrofit precisamos mapear os campos que queremos usar, sem muitos problemas, de forma prática.
* Não precisamos de anotações.
* Fazemos uma classe de configuração, colocar token, url base, etc.
* Programação declarativa. Escreve o suficiente pra ententer o que está acontecendo.
* Implementa a interface em tempo de execução, trazendo as informações da API. Dai precisamos ter o mapeamento dos dados.
